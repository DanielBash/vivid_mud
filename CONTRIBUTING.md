## Установка

```bash
git clone https://github.com/DanielBash/vivid_mud.git
cd vivid_mud
pip install -e .[dev]
```

---

## Тестирование

```bash
pytest -v
```

---

## Линтинг и форматирование

```bash
ruff check .
ruff format --check .
```

Чтобы автоматически исправить ошибки:

```bash
ruff check . --fix
ruff format .
```

## Воркфлоу

1. Сделайте форк репозитория
2. Создайте новую ветку:

   ```bash
   git checkout -b feature/my-feature
   ```
3. Внесите изменения
4. Убедитесь, что всё проходит:
   * тесты
   * линтинг

5. Сделайте коммит и отправьте ветку:

   ```bash
   git push origin feature/my-feature
   ```
6. Откройте PR

## Версии

Проект следует принципам SemVer:

* **MAJOR** — несовместимые изменения API
* **MINOR** — новая функциональность (обратно совместимая)
* **PATCH** — исправления багов

Пример:

```
0.2.1 → PATCH
0.3.0 → MINOR
1.0.0 → MAJOR
```
